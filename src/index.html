FAIR-DSM TOOL

Find your FAIR data level.

With the FAIR-DSM tool, you can quickly find the FAIR level of your data and work towards lowering barriers for long term data integration and re-use.
WHAT IS THE TOOL ABOUT?
Keeping the structure and representation of the dataset in mind, the tool is a simple questionnaire allowing users to find how much they need to standardize their data before sharing. Read more
HOW TO USE THE TOOL?
The tool asks you a number of question based on the data representation, metadata, and the hosting environment of the data.
Questionnaire format
The user will be asked questions on three broad aspects: Content, Representation, and Hosting facilities
Detail data inspection
Within each aspect, the user will be asked aspect specific questions and the user should select the best suitable answer(s).
Data maturity level
At the end, the user receive a maturity level report of their data enabling them to increase or compare pre-existing data aspects

Socialize with data experts

We understand the using the tool alone can sometime get confusing given the technicality of terms and details asked. Thus, we ask the users to always look around for data experts and use the tool within a group of people.

The recommended group of people would be data owners, legal experts (in case of dealing with sensitive data), data analyst, and data custodians or IT professionals. In case the user does not have all the above-mentioned people in the group, sufficient knowledge on how the data handling and data access is done should be there.
